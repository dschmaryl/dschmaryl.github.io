Projects

Fuel Efficiency Calculator

Android app to keep track of the fuel efficiency of your vehicles. Offers imperial and metrics units and the conversions to allow the user to switch back and forth. Show fuel efficiency as miles per gallon, liters per 100 kilometers, or even mixed units such as gallons per kilometer. View statistics such as average mpg, average cost per fill-up, etc. Try it out on Google Play.

BlackJack Trainer

A blackjack training app to practice basic strategy. Allows setting up table rules and includes the strategy chart which matches those rules. Includes stats for each possible hand, and allows the user to select which hands they want to practice. Built using React Native and Redux. Uses Firebase for analytics regarding app usage. To try it out, install it through Google Play.

Pomodoro Timer

A mobile app for using the Pomodoro technique. Created with React Native and Redux. Fully adjustable for work and break time lengths and alarm volume. Uses local push notifications when app is in the background. To try it out, install it through Google Play.

Golf-Stats

A web app for keeping track of rounds of golf played and calculating personal statistics. Created using the Flask framework; also uses Pandas for some of the data analysis. Currently working on building a new frontend with React.

i3 Layouts Scripts

Python scripts for managing tiled window layouts in the i3 window manager for Linux. Can be used to save tiled window layouts, open a saved layout, and also list or remove saved layouts.

Golf Handicap Estimator

Single-page web app which can give an estimate of a golfer's handicap index using a simplified version of USGA algorithm. Created using TypeScript, React and Redux.
